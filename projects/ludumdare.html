Ludum Dare

Ludum Dare is a game jam held multiple times each year where participants create games in a single weekend based around some theme. I participate in the more difficult compo, which disallows teams and mandates that every asset and line of code be created within 48 hours. Each submission is rated and ranked by other developers. Here are the games I've made for it:
